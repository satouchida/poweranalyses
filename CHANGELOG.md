# Changelog
All notable changes to this project will be documented in this file.

## [0.3.0] - 2026-04-28
### Added
- **Material 3 Design System**: Complete UI overhaul with Google's Material 3 principles.
- **Monte Carlo Simulation Engine**: Empirical power analysis using a dedicated Web Worker.
- **Interactive Power Curves**: Visual representation of statistical power using Chart.js.
- **Effect Size Calculator**: Dedicated tools for converting various statistical values to standardized effect sizes.
- **Enhanced Export Options**: Export results to CSV/TSV and added "Copy Table" functionality.
- **Improved Accessibility**: Implemented semantic HTML and ARIA labels for better screen reader support.

### Changed
- Refactored frontend logic for better modularity and performance.
- Optimized WebAssembly integration for statistical computations.

### Fixed
- Various layout issues in high-resolution displays.

## [0.2.1] - 2026-04-25
### Added
- Initial Tauri integration for native macOS support.
- Basic app icon support.

## [0.2.0] - 2026-04-24
- Initial release of the fork with vanilla JavaScript and basic Rmath integration.
